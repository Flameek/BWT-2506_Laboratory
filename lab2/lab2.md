# Лабораторная работа №2

## Тема
Функции в Python и базовые алгоритмы

## Цель работы
- Освоить принципы определения и использования функций в языке программирования Python.  
- Понять механизмы передачи аргументов в функции.  
- Научиться применять функции для решения практических задач.  
- Изучить базовые алгоритмические конструкции.

## Ход работы

### 1. Написание простых функций
- Реализованы функции `greet`, `square` и `max_of_two`.  
- Функция `greet` принимала имя пользователя и выводила приветствие.  
- Функция `square` возвращала квадрат переданного числа.  
- Функция `max_of_two` принимала два числа и определяла большее из них.

### 2. Работа с аргументами функций
- Создана функция `describe_person`, которая принимала имя и возраст пользователя.  
- Возраст был опциональным аргументом со значением по умолчанию 30.  
- Функция выводила информацию о человеке в читаемом виде.

### 3. Использование функций для решения алгоритмических задач
- Реализована функция `is_prime`, которая проверяла, является ли число простым.  
- Функция возвращала логическое значение `True` для простого числа и `False` для составного.  
- Проверки проводились для различных чисел, включая обработку некорректного ввода.

## Вывод
- Изучены и закреплены базовые принципы работы с функциями в Python.  
- Освоено использование аргументов и значений по умолчанию.  
- Получены навыки создания функций для решения алгоритмических задач.  
- Закреплено понимание структуры кода и повторного использования логических блоков.
