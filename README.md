# Лабораторные работы по курсу BWT-2506

**Автор:** Дьяконов Олег

## Описание
В данном репозитории собраны лабораторные работы по курсу BWT-2506, выполненные в течение первого семестра обучения.  
Работы охватывают ключевые темы курса и направлены на развитие практических навыков программирования на Python и работы с системами контроля версий.  

Основные темы, изученные и примененные в лабораторных работах:

- Работа с файлами в Python (чтение, запись, обработка исключений)  
- Использование функций и модулей для структурирования кода  
- Основы ООП: классы, объекты, методы, наследование, инкапсуляция, полиморфизм  
- Практическая работа с Git: создание репозиториев, клонирование, работа с ветками, коммиты, пуши и pull  
- Работа с WSL (Windows Subsystem for Linux) для выполнения команд Linux и тестирования скриптов  

Каждая лабораторная работа оформлена в отдельной папке (`lab1`, `lab2`, …, `lab9`) и содержит как исходный код на Python, так и отчёт в формате `.md`, в котором описаны цели, задания и выводы.

## Цель репозитория
Цель данного репозитория — систематизация всех лабораторных работ первого семестра по курсу BWT-2506 и демонстрация практических навыков программирования, а также работы с системой контроля версий Git и WSL.  

Репозиторий предназначен для:

- Повторения изученного материала  
- Демонстрации освоенных навыков работы с Python  
- Примера организации работы с лабораторными через Git, GitHub и WSL  

## Структура репозитория
- `lab1/` – работа с файлами, функции, обработка текста  
- `lab2/` – создание и использование модулей, практические задачи с функциями  
- `lab3/` – чтение и запись файлов, работа с исключениями  
- `lab4/` – сложные структуры файлов, пакеты и модули, импорт и использование собственных пакетов  
- `lab5/` – введение в ООП: классы, методы, работа с объектами  
- `lab6/` – инкапсуляция, приватные атрибуты, наследование  
- `lab7/` – множественное наследование, полиморфизм, управление командой сотрудников  
- `lab8/` – работа с WSL: выполнение команд Linux, тестирование Python-скриптов и взаимодействие с файловой системой  
- `lab9/` – практическая работа с Git: создание локальных и удалённых репозиториев, работа с ветками, коммиты и пуши  

## Примечание
Все временные файлы (`.pyc`, папки `__pycache__`) и ненужные файлы исключены из репозитория с помощью `.gitignore`.  
Каждая лабораторная работа снабжена подробным отчетом в формате Markdown, чтобы можно было легко проследить выполнение заданий и полученные результаты.
