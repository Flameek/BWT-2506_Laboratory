# Лабораторная работа №5

## Тема
Работа с классами

## Цель работы
- Получить практический опыт работы с объектно-ориентированным программированием (ООП) в Python.  
- Научиться создавать классы, методы и использовать конструкторы.  
- Освоить работу с атрибутами объектов и их модификацию.

## Ход работы

### 1. Базовый класс и методы
- Создан класс `Book` с атрибутами: `title` (название), `author` (автор), `year` (год издания).  
- Добавлен метод `get_info()`, который возвращает информацию о книге в формате:  
  `"Название книги: [title], Автор: [author], Год издания: [year]"`.  
- Разработан бесконечный цикл для добавления книг пользователем с проверкой корректности ввода:  
  - Название книги не может быть пустым и содержать менее двух символов.  
  - Имя автора также проверяется на пустоту и минимальную длину.  
  - Год издания проверяется на допустимый диапазон (1000–2025) и на корректность числового ввода.  
- Все добавленные книги сохраняются в списке `books` и отображаются пользователю.

### 2. Работа с конструктором
- Создан класс `Circle` для представления круга.  
- Использован конструктор `__init__` для инициализации радиуса (`radius`).  
- Добавлены методы:  
  - `get_radius()` — возвращает значение радиуса.  
  - `set_radius(new_radius)` — позволяет изменить радиус с проверкой корректности значения.  
- Реализован цикл ввода радиуса с проверкой на отрицательные значения и некорректный ввод.  
- Пользователь может изменить радиус объекта, после чего выводится новый радиус на экран.

- Получен опыт реализации проверки корректности пользовательского ввода при работе с объектами.  
- Практика показала важность использования методов для безопасного изменения состояния объектов.
