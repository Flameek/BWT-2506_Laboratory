# Лабораторная работа №6

## Тема
Работа с классами. Инкапсуляция и наследование

## Цель работы
- Получить практический опыт работы с объектно-ориентированным программированием (ООП) в Python.  
- Научиться использовать инкапсуляцию для защиты данных.  
- Освоить наследование и полиморфизм в классах Python.

## Ход работы

### 1. Защита данных пользователя
- Создан класс `User_Account` с атрибутами:  
  - `username` — имя пользователя  
  - `email` — электронная почта  
  - `__password` — приватный атрибут для хранения пароля  
- Использован конструктор `__init__` для инициализации атрибутов.  
- Реализованы методы:  
  - `set_password(new_password)` — безопасно изменяет пароль аккаунта.  
  - `check_password(password)` — проверяет введённый пароль, возвращает `True`, если пароль верный, иначе `False`.  
- Реализован цикл ввода данных пользователя с проверкой корректности.  
- На практике проверена возможность изменения пароля и его проверки через методы класса.

### 2. Полиморфизм и наследование
- Создан базовый класс `Vehicle` с атрибутами `make` (марка) и `model` (модель).  
- Метод `get_info()` возвращает информацию о транспортном средстве.  
- Создан класс `Car`, наследующий `Vehicle`, добавлен атрибут `fuel_type` (тип топлива).  
- Метод `get_info()` переопределён так, чтобы включать информацию о типе топлива.  
- На практике создан объект `Car`, введены данные пользователем, выведена информация с использованием переопределённого метода.

## Вывод
- Освоена инкапсуляция и использование приватных атрибутов для защиты данных.  
- Получен опыт работы с наследованием и полиморфизмом в Python.  
- На практике реализованы безопасные операции с пользовательскими данными и расширение функционала базового класса через наследование.
