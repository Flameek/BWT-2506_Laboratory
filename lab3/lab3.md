# Лабораторная работа №3

## Тема
Работа с файлами в Python: открытие, чтение, запись, работа с исключениями

## Цель работы
- Освоить работу с файлами в языке Python.  
- Научиться открывать, читать и записывать файлы.  
- Изучить обработку исключений при работе с файлами.  
- Применять функции для решения практических задач с файловыми данными.

## Ход работы

### 1. Открытие и чтение файла
- Создан текстовый файл `example.txt` с несколькими строками текста.  
- Разработана функция для чтения файла в двух режимах:  
  - чтение всего файла сразу (`all`),  
  - построчное чтение (`line`).  
- Реализована проверка существования файла с обработкой исключения `FileNotFoundError`.  
- На практике было продемонстрировано корректное чтение файла и обработка отсутствующих файлов.

### 2. Запись в файл
- Написана программа, которая запрашивает у пользователя текст и записывает его в новый файл `user_input.txt`.  
- Реализована возможность добавления текста в существующий файл без удаления его предыдущего содержимого.  
- Продемонстрированы варианты записи с режимами `w` (создание/перезапись файла) и `a` (добавление текста).

### 3. Работа с исключениями
- Функции модифицированы для корректной обработки ошибок при открытии и записи файлов.  
- Использован блок `try-except` для перехвата ошибок типа `FileNotFoundError` и `PermissionError`.  
- Программа выводит понятное сообщение пользователю при возникновении исключений.

## Вывод
- Изучены базовые операции работы с файлами в Python: открытие, чтение, запись.  
- Освоена обработка исключений, возникающих при работе с файлами.  
- Получены навыки структурирования кода с использованием функций для файловых операций.  
- Применение разных режимов чтения и записи позволяет гибко работать с файлами и предотвращать потерю данных.
