# Лабораторная работа №4

## Тема
Модули и пакеты: импорт, создание, использование

## Цель работы
- Изучить принципы создания и использования модулей и пакетов в Python.  
- Научиться импортировать стандартные модули (`math`, `datetime`).  
- Освоить создание собственного модуля и пакета с функциями.  
- Применять модули и пакеты для структурирования кода и повторного использования функций.

## Ход работы

### 1. Импорт стандартных модулей
- Импортированы модули `math` и `datetime`.  
- Использован `math.sqrt()` для вычисления квадратного корня числа.  
- С помощью `datetime.datetime.now()` получена текущая дата и время.  
- Реализована проверка корректности ввода числа с обработкой исключений.

### 2. Создание и использование собственного модуля
- Создан модуль `my_mudule.py` с функцией `summa(a, b)`, возвращающей сумму двух чисел.  
- Импорт модуля `my_mudule` выполнен в другом файле и использован для вычисления суммы двух пользовательских чисел.  
- Обеспечена обработка некорректного ввода чисел с помощью блока `try-except`.

### 3. Создание и использование пакетов
- Создан пакет `my_package` с модулями: `ln`, `log10`, `log_osn`, `summ`, `sq`, `stepen`.  
- Каждый модуль выполняет отдельную математическую функцию (логарифмы, сложение, квадратный корень, возведение в степень).  
- Разработана программа-математический помощник, которая:  
  - позволяет выбрать раздел (`логарифмы` или `математические действия`),  
  - применяет функции из пакетов для выполнения выбранных операций,  
  - проверяет корректность ввода пользователя.  
- Использован `sys.path.append()` для корректного импорта пакетов внутри скриптов.

## Вывод
- Освоено использование стандартных модулей Python для вычислений и работы с датой и временем.  
- Научились создавать собственные модули и импортировать их для решения задач.  
- Создан пакет с набором функций, что позволило структурировать проект и сделать код повторно используемым.  
- Получен практический опыт построения интерактивных программ с модулями и пакетами.
